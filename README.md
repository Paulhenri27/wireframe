# Wireframe

A simple project for hosting single HTML files.

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v14 or higher)

### Installation

```bash
npm install
```

### Usage

Start the server:

```bash
npm start
```

The server will start at `http://localhost:8080`.

### Development

For development with cache disabled:

```bash
npm run dev
```

## Adding HTML Files

Simply add your HTML files to the root directory and they will be served automatically.

For example:
- `index.html` → `http://localhost:8080/`
- `about.html` → `http://localhost:8080/about.html`
